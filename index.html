Tabelas

Para incorporar os arquivos de áudio em nossa pagina web utilizamos a TAG áudio :

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação e o layout de documentos HTML. Ele permite personalizar elementos como fontes, cores, espaçamento, bordas, alinhamento, entre outros. Com CSS, é possível estilizar não apenas o conteúdo HTML, mas também elementos como tabelas, listas, parágrafos e até mesmo linhas e bordas. Comcumento na pasta do nosso projeto, que deverá conter a extenção .css. Como este é um arquivo diferente do html precisamos realizar a conexão entre eles assim nao exemplo ficam as estilizações que realizamos nesta página web. JavaScript (JS) é uma linguagem de programação utilizada principalmente para adicionar interatividade e dinamismo a páginas web. Com o JS, é possível criar efeitos visuais, validar formulários, manipular elementos da página, criar animações e até mesmo realizar requisições assíncronas para atualizar conteúdo sem recarregar a página. Para tanto precisamos criar um novo documento na pasta do nosso projeto, que deverá conter a extenção .js.
